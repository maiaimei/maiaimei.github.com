# 重置密码

原文链接：https://blog.csdn.net/qq_44895681/article/details/120530904

官方文档：https://docs.gitlab.com/ee/security/reset_user_password.html

  1. 重置密码之前，需先使用root用户登录到gitlab所在服务器。

    gitlab-rails console -e production
    1

  2. 等待控制台加载完毕，有多种找到用户的方法，您可以搜索电子邮件或用户名。

    user = User.where(id: 1).first
user = User.find_by(email: 'admin@example.com')
    1
    2

  3. 现在更改密码。注意，必须同时更改密码和password_confirmation才能使其正常工作。

    user.password = '新密码'
user.password_confirmation = '新密码'
    1
    2

  4. 保存更改。

    user.save
    1
