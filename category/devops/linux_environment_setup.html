VMware Workstation、Centos7及Ubuntu18安装与配置

maiaimei 2021/12/1 Linuxcentosubuntu

# VMware安装

下载 VMware Workstation Pro | CN (opens new window)

如果是最小化安装，则需要安装以下工具：

yum install -y net-tools
yum install -y vim
1
2

如果虚拟机无法ping通宿主机，在控制面板-系统和安全-Windows Defender 防火墙，点击高级设置，选择入站规则，把虚拟机监控(回显请求-ICMPv4-In)的规则启用即可。

# Centos7安装及配置

VMware安装Centos7超详细过程 (opens new window)

# 配置静态IP和主机名

# 修改IP
cd /etc/sysconfig/network-scripts
vim ifcfg-ens33

# 添加或修改以下内容
TYPE="Ethernet"
BOOTPROTO="static"
ONBOOT="yes"
IPADDR=192.168.1.10
GATEWAY=192.168.1.1
NETMASK=255.255.255.0
DNS1=114.114.114.114
DNS2=8.8.8.8

# 修改主机名
hostnamectl set-hostname 主机名

# 重启
reboot

# 检查IP
ifconfig
ping baidu.com
ping 宿主机

# 检查主机名
hostname

# 配置静态IP

修改网卡配置文件

vim /etc/network/interfaces
1

添加

# interfaces(5) file used by ifup(8) and ifdown(8)
auto lo
iface lo inet loopback

auto ens33
iface ens33 inet static
address 192.168.1.29
netmask 255.255.255.0
gateway 192.168.1.1

# 配置DNS域名解析，解决ping baidu.com不通

修改DNS配置文件

vim /etc/systemd/resolved.conf
1

添加

[Resolve]
DNS=8.8.8.8 114.114.114.114
1
2

重启域名解析服务

systemctl restart systemd-resolved
systemctl enable systemd-resolved

# 使用Open VM Tools安装VMware Tools

在Ubuntu服务器安装VMware Tools

apt install open-vm-tools -y
1

在Ubuntu桌面版安装VMware Tools

apt install open-vm-tools-desktop -y
1

开启VMware Tools服务

systemctl start open-vm-tools.service
1

设置开机启动

systemctl enable open-vm-tools.service
1

挂载共享文件夹，打开/etc/fstab文件

vim /etc/fstab
1

插入一行

.host:/         /mnt/hgfs               fuse.vmhgfs-fuse allow_other,defaults   0       0
1

重启系统后进去文件夹

cd /mnt/hgfs
1

可以看见宿主机共享出来的文件夹
