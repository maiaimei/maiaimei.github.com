在Docker下安装RabbitMQ

maiaimei 2022/8/1 RabbitMQ

# 拉取镜像

拉取后缀带management的镜像，这种镜像才有Web管理界面。

docker pull rabbitmq:3.10.6-management
1

# 运行镜像

docker run -d \
--name some-rabbit \
--hostname my-rabbit \
-p 5672:5672 -p 15672:15672 \
-v /opt/rabbitmq:/var/lib/rabbitmq \
-e RABBITMQ_DEFAULT_USER=guest \
-e RABBITMQ_DEFAULT_PASS=guest \
--restart always \
rabbitmq:3.10.6-management
