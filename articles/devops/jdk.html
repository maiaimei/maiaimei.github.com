JDK安装与配置

maiaimei 2021/10/9 JavaJDK

# 下载地址

  • https://www.oracle.com/java/technologies/downloads/ (opens new window)
  • https://www.oracle.com/java/technologies/downloads/#java8 (opens new window)

# 参考手册

  • JDK 17 Documentation (opens new window)
  • JDK 8 Documentation (opens new window)

# 安装配置

# 在Windows下安装配置

  1. 下载[jdk-8u321-windows-x64.exe (opens new window)]

  2. 双击运行，安装到指定路径

  3. 配置环境变量

​ 3.1 新建 JAVA_HOME 环境变量，指向JDK安装目录

​ 3.2 编辑 Path 环境变量，添加 %JAVA_HOME%\bin

多版本JDK，配置 JAVA_8_HOME、JAVA_17_HOME，将 JAVA_HOME 指向 %JAVA_X_HOME%

如果是 JDK8 ，Path环境变量需要添加 %JAVA_HOME%\bin 和 %JAVA_HOME%\jre\bin

执行 java -version 结果可能不是我们预期的，这是因为在 JDK 安装时会自动将 Java.exe，javaw.exe，javaws.exe 复制到 C 盘，可以在 cmd 中使用 where java 查看它们的位置并删除，然后重新打开一个cmd，执行 java -version

# 在Linux下安装配置

  1. 下载jdk-8u321-linux-x64.rpm (opens new window)或jdk-8u321-linux-x64.tar.gz (opens new window)，并上传到/usr/local

  2. 卸载JDK

    # 查看是否已经安装jdk
rpm -qa | grep jdk

# 卸载默认安装的jdk
rpm -e --nodeps jdk名称

rpm -e --nodeps copy-jdk-configs-3.3-10.el7_5.noarch
rpm -e --nodeps java-1.8.0-openjdk-headless-1.8.0.262.b10-1.el7.x86_64
rpm -e --nodeps java-1.8.0-openjdk-1.8.0.262.b10-1.el7.x86_64
rpm -e --nodeps java-1.7.0-openjdk-1.7.0.261-2.6.22.2.el7_8.x86_64
